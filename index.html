FAQ

Important questionsأسئلة مهمة

Can I buy and download PhantomKit today?هل أستطيع شراء وتحميل PhantomKit اليوم؟

No customer binary is available yet. Code signing remains the release blocker, and unsigned builds are not distributed.لا توجد نسخة عميل متاحة حالياً. التوقيع الرقمي ما زال بوابة الإطلاق، ولا تُوزع النسخ غير الموقعة.

Does it guarantee complete vulnerability detection?هل يضمن اكتشاف جميع الثغرات؟

No. Findings reflect configured scope, profile, environment, and assessment time. An absence of findings is not proof of security.لا. النتائج تتأثر بالنطاق والإعداد والبيئة ووقت التقييم، وعدم وجود نتيجة لا يثبت أن النظام آمن.

Where is customer data stored?أين تُخزن بيانات العميل؟

PhantomKit is designed to run on customer-controlled infrastructure. Customer data custody is not taken by default, and optional report intelligence defaults off.صُمم PhantomKit ليعمل على بنية يتحكم بها العميل. لا تُؤخذ حيازة بيانات العميل افتراضياً، وذكاء التقارير الاختياري متوقف افتراضياً.

Can it be used for client assessments?هل يمكن استخدامه لتقييم أنظمة العملاء؟

Only under an applicable commercial license and explicit written authorization for every target and assessment window.فقط بترخيص تجاري مناسب وتفويض كتابي صريح لكل هدف ونافذة تقييم.

Is standards mapping a compliance certification?هل ربط المعايير شهادة امتثال؟

No. Mapping is advisory reporting alignment and never a legal, regulatory, or audit certification.لا. الربط إرشادي لتنظيم التقارير وليس شهادة قانونية أو تنظيمية أو نتيجة تدقيق.
